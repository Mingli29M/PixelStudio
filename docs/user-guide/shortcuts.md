# Keyboard Shortcuts

Master PixelStudio's keyboard shortcuts to speed up your pixel art workflow.

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New project |
| `Ctrl+O` | Open project |
| `Ctrl+S` | Save project |
| `Ctrl+Shift+S` | Save as |
| `Ctrl+W` | Close project |
| `Ctrl+Q` | Quit application |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` or `Ctrl+Shift+Z` | Redo |
| `Ctrl+X` | Cut |
| `Ctrl+C` | Copy |
| `Ctrl+V` | Paste |
| `Delete` or `Backspace` | Delete selection |
| `Ctrl+A` | Select all |
| `Ctrl+D` | Deselect |

## Custom Shortcuts

You can customize shortcuts in **Edit > Preferences > Keyboard Shortcuts**.

**Tips for Custom Shortcuts:**
- Choose intuitive combinations
- Avoid conflicts with system shortcuts
- Use modifiers (Ctrl, Shift, Alt) for complex actions
- Keep frequently used actions easily accessible
- Export/import shortcut profiles for different workflows

## Tips for Efficient Workflow

1. **Learn Tool Shortcuts First**: Master P, E, G, L, R, C for basic drawing
2. **Use Quick Color Picking**: Alt+Click is faster than switching to eyedropper
3. **Space+Drag for Pan**: Essential for navigating large canvases
4. **Bracket Keys**: Quick brush size adjustment without leaving the canvas
5. **Layer Navigation**: Alt+[ and Alt+] for rapid layer switching
6. **Frame-by-frame**: Use , and . to scrub through animation frames
7. **Workspace Switching**: Alt+1/2/3 for different task layouts
8. **Hold Shift**: Constrain tools to straight lines or perfect shapes

## Platform-Specific Notes

### macOS
Replace `Ctrl` with `Cmd` for most shortcuts:
- `Cmd+C` for Copy
- `Cmd+V` for Paste
- `Cmd+Z` for Undo

### Linux
Standard `Ctrl` shortcuts work as documented.
